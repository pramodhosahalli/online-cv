Here are some of my projects

Explain an Image using Deep Learning and LSTM (Python)

This project Explains an Image Using Deep Learning Flickr 30K Images along with captions are used as input dataset Features of Images are obtained using Pre-trained Model (InceptionResNetV2). Captions are preprocessed and then fed along with features to train our Deep Learning model.Captions are Sequentially generated using LSTM. Google Colab used for Development

Suspecious Transaction Detection using machine Learning

Credit card fraud dataset is obtained from kaggle which has 31 parameters Correlation of each feature with output (class) is observed. Considering only outliers based on correlation, by choosing equal number of regular and fraud transactions, model is trained. Based on training suspicious transactions are detected.(Classification classes used are SVM, Random Forest Classifier, Logistic Regression)

Chat Application using Pub nub API (Angular JS and C#)

Chat Application using Angular JS and C# and Pub nub API. Pub nub allows to create a channel and users can subscribe to a channel based on subscribe key and user can publish to a channel using publish key. Chat application provides private chat, public chat, Group chat etc.

Rally Chatbot Using Gupshup API (Sponsoured by Cisco)

Gupshup API allows creation of bot template.Bot written using Java,which communicates with Rally for task creation,defect creation,updation, deletion,querying a defects,closing defects etc. Chatbot could be used on Whatsapp, Telegram, Cisco spark etc..

Sentimental Analysis using Twitter API

For the given location and for given person, tweets are read using Twitter API using hash tags. Tweets are stemmed for better results. Tweets are categorized into positive, negative or neutral. Based on the counts of positive, negative and neutral values decisions are predicted

Text Summarization

Text Summarization, a java web application which accepts the large text and summarizes the text with given number of lines based on the given threshold. Sequence of text given as input. Each sentence compared with other sentences are redundant sentences are removed based on the given threshold. Stemming are done for words, for better results.
